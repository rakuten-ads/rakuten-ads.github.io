Open SPS Portal UI

Receiver

RakutenReward

Parameters

rz

Rz Cookie value. @since 7.3.0, Rz Cookie is mandatory for personalized experience in SPS Home Screen

isPortalOpenedCallback

Callback to get the result of whether SPS portal is opened or not.

activityResultCallback

ActivityResultCallback to be triggered when SPS portal is closed.

Sample to handle closed event

RakutenReward.openSpsPortal({ result ->
   when (result) {
      is Failed -> // Failed to open Portal. Get the error here `result.error`
      is Success -> // SDK Portal opened successfully
  }
}) {
    // handle portal closed event
}
