RakutenRewardJava

object RakutenRewardJava

Java-compatible APIs for RakutenReward SDK

This class provides Java-style callback interfaces for Android projects written in Java that need to use Rakuten Reward SDK functionality.

This is a wrapper around the core RakutenReward APIs, providing the same functionality with a more Java-friendly interface.
